## 🛠️ Tech Stack Used

| Layer | Technology |
|-------|-----------|
| Frontend | React + Vite + Tailwind CSS + @dnd-kit |
| Backend | Node.js + Express + SQLite |
| Auth | JWT |
| Notifications | Telegram Bot + Email (Nodemailer) |
| Deployment | Systemd + Cloudflare Tunnel |
| AI Assistant | OpenClaw + GLM-4 |
